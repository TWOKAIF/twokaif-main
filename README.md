## 5. Важные технические детали

- **Шрифт:** Inter Tight (300–700, italic 400) через Google Fonts
- **Анимации:** GSAP 3.12.5 + ScrollTrigger (cdnjs)
- **Lenis:** подключён но отключён (нативный скролл)
- **Курсор:** кастомная снежинка с `mix-blend-mode: difference`, скрыта на тач-устройствах
- **Прелоадер:** имеет `window.restartLoader()` — вызывается при клике на лого
- **Снежинка SVG:** единый viewBox `-3 18 81 81` во всех блоках
- **Цветовая схема:** текст `#1d1d1f`, muted `#86868b`, фон `#ffffff`, surface `#f5f5f7`
- **Отступы:** `--edge` 48/32/24/20/16px по брейкпоинтам 1200/960/640/480
- **Typography:** Apple-style clamp() с брейкпоинтами
