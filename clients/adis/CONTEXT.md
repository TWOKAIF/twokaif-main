## Workflow для новой сессии

1. Проверить `claude mcp list` — убедиться что `framer: ✓ Connected`
2. Через Framer MCP прочитать структуру Palmer (copy):
   - getProjectXml — общая структура
   - getNodeXml — детали по нодам
   - getCMSCollections — Work и Article коллекции
3. Адаптировать тексты под Адиса:
   - Hero «Akihiko™» → «Адис Маммо»
   - Sub-text «13+ years of digital form...» → био Адиса
   - CMS Work: 5 кейсов sonder-goods/halo-wear/... → реальные кейсы Адиса (свадьбы со звёздами)
4. Подменить фото
5. Опубликовать через Framer Publish или экспортировать код для Aeza
